# CM5 Unified Carrier

Carrier board for Radxa CM5 used in the Hyper-K camera system.

Features:
- 48V input
- PoE option
- USB3 camera interface
- Ethernet
- Uses drop in DC/DC modules for 48V->12V (ext power mode)
- Uses drop in DC/DC module for 12V->8V (IWCD camera)
- Uses drop in DC/DC module for 12V->5V (both cameras)
